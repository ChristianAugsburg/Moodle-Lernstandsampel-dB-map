# Moodle-Lernstandsampel-dB-map
🚦 Moodle Lernstands-Ampel (Widget) Ein leichtgewichtiges, JavaScript-basiertes Widget für Moodle, das den Lernfortschritt aus einer Lernlandkarte visualisiert und direkt in ein Datenbank-Modul zurückschreibt. Das Widget bietet Schülern eine schnelle Status-Rückmeldung (🔴, 🟠, 🟢) und Lehrkräften eine kompakte Auswertung direkt im Kurs-Interface.
